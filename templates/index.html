Overview

The Movies Informatics project is a data visualization project that provides an overview of movie genres from 1916 to 2017. The data for this project was obtained from Kaggle and plotted in a visual manner, allowing users to easily compare and analyze the trends and patterns of the movie industry over the years. This project aims to provide valuable insights into the evolution of movie genres and the factors that influence their success, as well as to serve as a useful resource for researchers, movie enthusiasts, and industry professionals alike.

Tools & Technology

Below are some of the tools and technologies used to create this repository:

  • Python
  • JavaScript
  • MongoDB
  • HTML

Biases & Further Considerations

Some factors that we should consider when analyzing this dataset.

  • Revenue currency is not provided
  • Data quality analysis has not been performed
  • Missing budgets were provided a value of zero
  • Selection bias: dataset is not a full depiction of all movies (i.e., independent films)

Acknowledgements

The dataset was obtained from Kaggle, which had been generated from The Movie Database API. A further thanks goes to all the instructors through the University of Toronto Data Analytics program.
