Movie Informatics / Movies Info

This repository summarizes and compares the genres of movies over years in a visual manner. We obtained data from kaggle and created the plots based on yearly data for movies from 1916 to 2017. Some of the criteria summarized are genres, budget, revenue, voter counts and production countries.

Tools & Technology

  • JavaScript
  • MongoDB
